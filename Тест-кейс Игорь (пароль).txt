Тест кейс №1

Название : Проверка строки "Пароль".

Описание: Проверка строки "Пароль" на соответствие требованиям : Латиница, спецсимволы, заглавная буква

Шаги выполнения:
1)Введение символов латинницей в сочетакнии с спец символами и несоклькими загланвыми буквами в установленной границе (3-15 символов)
 Нажми кнопку отправить

Ожидаемый результат: Форма отправилась (не появилось предупреждение о неправильном пароле)

--------
2)Введение только латинницы
 Нажми кнопку отправить

Ожидаемый результат: Форма отправилась (не появилось предупреждение о неправильном пароле)


--------
3)Введение только спец символов
 Нажми кнопку отправить

Ожидаемый результат: Форма отправилась (не появилось предупреждение о неправильном пароле)


--------
Ожидаемый результат: Возможно введение только вариантов соответствующих условиям.
Фактический результат: Строка не принимает латинницу.
Статус: Тест не пройден

Тест кейс №2

Строка пароль

Описание: Проверка строки "Пароль" на корректность соблюдения границ ввода символов

Шаги выполнения:

1)Введение 3 символов
 Нажми кнопку отправить

Ожидаемый результат: Форма отправилась (не появилось предупреждение о неправильном пароле)
Фактический результат:Форма не отправилась ( появилось предупреждение о неправильном пароле)


2)Введение 15 симоволов
 Нажми кнопку отправить
Ожидаемый результат: Форма отправилась (не появилось предупреждение о неправильном пароле)
Фактический результат: Форма отправилась (не появилось предупреждение о неправильном пароле)


Негативный тест кейс :
Введение 16 символов
 Нажами кнопку отправить

Ожидаемый результат: Допустимое значение введеных символов от 3 до 15.
Фактический результат: Нижняя граница больше 3 , верхняя граница больше 15.
Статус: Тест не пройден
